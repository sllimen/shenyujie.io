Javascript 跨域(六)之 document.domain + iframe

Aug 21, 2016 | javascript | Hits

对于主域相同而子域不同的情况，可以通过设置 document.domain 的办法来解决。
具体的做法我们可以在 http://www.shenyujie.com/a.html 和 http://m.shenyujie.com/b.html 下面共同设置 document.domain = ‘shenyujie.com’，然后在 a.html 下面创建一个隐藏的 iframe，去控制 iframe 的contentDocument，这样我们就可以实现交互了
( 这两个域名必须属于同一个基础域名!而且所用的协议，端口都要一致，否则无法利用document.domain进行跨域 )
