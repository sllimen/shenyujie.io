Aug 28, 2016 | javascript | Hits

前言：终于写到跨域的倒数第二篇了（最后一篇flash因为兼容性太奇葩还要额外装插件，我们先无视），算下来这可以算是最后一篇跨域的文章也不为过。
既然写文章，肯定是要百度一番的，何况我对 CORS 跨域用的不算多，不是很熟练。上百度一搜，这么重要的前端跨域技术竟然很少有提及，卫星定位系统反而占据了搜索页的第一条，我不得不感叹，果然度娘只适合看各种内涵段子，技术问题还是要问 google

优势

  1. CORS 支持所有类型的 HTTP 请求
  2. 开发者可以使用普通的 XMLHttpRequest 来发起或是取得数据
  3. 现在大部分主流的浏览器均以支持 CORS
